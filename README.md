# Origins: Classes (NeoForge)

This is a NeoForge port for [Origin: Classes](https://github.com/apace100/origins-classes), worked
with [Origins (NeoForge)](https://github.com/IAFEnvoy/Origins-NeoForge).

**This mod is still in early development. If you meet any bugs please report on GitHub**

## Description

This mod is an add-on to the Origins mod. In addition to choosing an origin, you now also choose a class.
Classes available are the following:

* Archer
* Beastmaster
* Blacksmith
* Cleric
* Cook
* Explorer
* Farmer
* Lumberjack
* Merchant
* Miner
* Nitwit
* Rancher
* Rogue
* Warrior

## FAQ

### This mod is not working in my game.

Make sure you are using Origins (NeoForge), you can download it from
both [CurseForge](https://www.curseforge.com/projects/1375372) and [Modrinth](https://modrinth.com/project/QaMa6tXQ).
Also, make sure you have the latest version of both mods.

If still not working, please report on GitHub with your log file.

### Some functions don't work like what on Fabric?

To capable with NeoForge Platform, I make some changes to the original code, so some functions may not work like what on
Fabric version.
